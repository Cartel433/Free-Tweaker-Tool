## 📖 Overview

**NZSV Collabed App** is a Python-based optimization tool designed for **gamers** who want:
- 🎮 **Lower latency** (reduce input delay)
- 📈 **Higher FPS** (smoother gameplay)
- 🧹 **Clean temp files** (free up disk space)
- ⚡ **Better system performance**

If your PC runs at "shit speeds" — this app fixes that.

---

## ✨ Features

| Category | What it does |
|----------|---------------|
| **Low Latency Mode** | TCP/IP optimizations, disable Nagle's algorithm |
| **FPS Boost** | Disable background apps, game mode tweaks |
| **Temp Cleaner** | Remove Windows temp, browser cache, prefetch |
| **DNS Flush** | Clear DNS cache for faster connections |
| **Registry Tweaks** | Gaming-optimized registry settings |

## 🚀 How to Use

### Prerequisites
- Windows 10 or 11
- Python 3.8 or higher
- **Run as Administrator** (required for system tweaks)

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/Cartel433/NZSV-Collabed-App

# 2. Go into the folder
cd NZSV-Collabed-App

# 3. Install dependencies (optional, for colors)
pip install colorama

# 4. Run the app
python "Tweak App.py"
